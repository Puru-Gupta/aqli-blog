# Uploads

Put blog assets here:

- `.png`, `.jpg`, `.jpeg`, `.gif`, `.svg`, `.webp` for figures and maps.
- `.pdf` for reports or appendices.
- `.csv`, `.xlsx`, `.xls`, `.tsv` for tables.
- `.zip` for bundles.

Use lowercase filenames with hyphens, for example:

```text
india-district-pm25-2023.png
annual-update-appendix.pdf
district-results.csv
```
