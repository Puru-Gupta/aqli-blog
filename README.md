# AQLI Blog Portal

A professional GitHub Pages blog template for AQLI-style research posts, policy explainers, charts, and downloadable files.

## What is included

- GitHub Pages/Jekyll setup.
- Responsive homepage, archive, about page, and publishing guide.
- Markdown post template for non-coders.
- Captioned figure and downloadable file includes.
- Upload folder for PNG/JPG/PDF/CSV/XLSX/ZIP assets.
- Client-side search for posts.

## Publish

Read [PUBLISHING.md](PUBLISHING.md) or open `/publish/` on the site.

## Local preview

If Ruby/Jekyll is installed:

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000`.

GitHub Pages will build the site automatically even if you do not preview locally.
